# Proof-of-Work CAPTCHA Research

This repository contains research and implementations related to proof-of-work (PoW) based CAPTCHAs, exploring their effectiveness in preventing automated abuse while maintaining user experience.

## Overview

Proof-of-work CAPTCHAs leverage computational puzzles to distinguish human users from bots. This project investigates various PoW algorithms, their integration with web applications, and performance evaluations.
